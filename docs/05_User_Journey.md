# User Journey

## Purpose

This document describes the end-to-end user journey for AI Investment Research Copilot, illustrating how users interact with the platform to research a company and generate AI-assisted investment insights.

---

# User Journey Overview

```text
Login

↓

Search Company

↓

Retrieve Financial Data

↓

Analyze Financial Statements

↓

Analyze Technical Indicators

↓

Analyze News & Sentiment

↓

Generate Investment Research Report

↓

Review AI Insights

↓

Make Investment Decision
```

---

# Step 1: Login

### User Action

The user signs in to the platform.

### System Response

- Authenticates the user.
- Displays the dashboard.
- Provides access to the company search.

---

# Step 2: Search Company

### User Action

The user searches for a publicly listed company (e.g., Apple, Microsoft, Reliance Industries).

### System Response

- Retrieves company information.
- Displays current market data.
- Shows available financial reports and recent news.

---

# Step 3: Financial Analysis

### User Action

The user requests a financial analysis.

### System Response

The platform analyzes:

- Income Statement
- Balance Sheet
- Cash Flow Statement
- Financial Ratios
- Revenue Growth
- Profit Growth
- Debt Levels

---

# Step 4: Technical Analysis

### User Action

The user requests technical analysis.

### System Response

The platform evaluates:

- RSI
- MACD
- Moving Averages
- Bollinger Bands
- Support & Resistance
- Trading Volume
- Price Trends

---

# Step 5: News & Sentiment Analysis

### User Action

The user requests the latest company developments.

### System Response

The platform summarizes:

- Company news
- Industry news
- Earnings announcements
- Significant events
- Overall sentiment

---

# Step 6: AI Investment Research Report

### System Generates

A structured report containing:

- Company Overview
- Financial Health
- Technical Analysis
- News Summary
- Key Opportunities
- Key Risks
- Confidence Score
- Overall Investment Outlook

---

# Step 7: Human Review

### User Action

The user reviews the AI-generated report.

### User Can

- Validate findings
- Read supporting evidence
- Compare insights with external research
- Export the report for future reference

---

# Step 8: Investment Decision

The user independently decides whether to:

- Continue researching
- Monitor the company
- Discuss with an advisor
- Make an investment decision

The platform does **not** execute trades or provide personalized financial advice.

---

# User Experience Goals

The platform should enable users to:

- Complete research faster
- Reduce manual effort
- Understand financial information more easily
- Receive explainable AI insights
- Make informed investment decisions

---

# Design Principles

The user journey is designed to:

- Minimize unnecessary steps.
- Present information in a logical sequence.
- Provide transparent AI reasoning.
- Keep users in control of final decisions.
- Support efficient investment research.

---

# Success Criteria

A successful user journey enables users to:

- Find relevant company information quickly.
- Understand financial performance.
- Interpret technical indicators with confidence.
- Stay informed about recent developments.
- Generate consistent investment research reports.
- Make well-informed investment decisions.
